🤖 Sistema de Automação de Cadastros

Automatiza o processo manual de login e cadastro de produtos em uma plataforma web a partir de dados estruturados em arquivos locais. O sistema simula as ações humanas na interface do usuário (cliques, digitação e rolagem), incluindo mecanismos de tolerância a falhas para não interromper a execução caso ocorra um erro em um item específico.

✅ Problema resolvido: Elimina o trabalho manual repetitivo, reduz drasticamente o tempo gasto em inserção de dados, anula erros humanos de digitação e garante a continuidade do processo através de logs detalhados e tratamento de exceções.

🛠️ Tecnologias: Python · PyAutoGUI · Pandas · Pyperclip · Logging · Time

What it does
Automates the manual process of login and product registration on a web platform using structured data from local files. Simulates human actions on the UI (clicks, typing, and scrolling), with built-in fault tolerance to keep the process running even if individual items fail.
What it solves
Eliminates repetitive manual work, drastically reduces data entry time, eliminates human typing errors and ensures process continuity through detailed logs and exception handling.

🤖 Registration Automation System

Automates the manual process of login and product registration on a web platform using structured data from local files. The system simulates human actions on the user interface (clicks, typing, and scrolling), with fault-tolerance mechanisms to prevent execution from stopping when an error occurs on a specific item.

✅ Problem solved: Eliminates repetitive manual work, drastically reduces the time spent on data entry, eliminates human typing errors, and ensures process continuity through detailed logs and exception handling.
